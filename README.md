# trajectory-project-2026

_Оценка траектории обучения:_ 

У вас есть полная история всех оценок студентов крупного университета за несколько лет, включая дисциплины по выбору. Для некоторых дисциплин (очных), доступны также данные о посещаемости. Кроме того, доступны данные СОП. Создайте модель, которая будет предсказывать успешность освоения конкретным студентом выбранных дисциплин

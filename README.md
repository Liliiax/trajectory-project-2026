# Оценка траектории обучения

## Описание проекта
Trajectory Project (?) Оценка траектории обучения $-$ веб-приложение для прогнозирования успешности освоения студентом выбранной дисциплины на основе его образовательной траектории. 

## Возможности

- **Предобработка данных** $-$ автоматическая подготовка входных данных.
- **Прогноз итоговой оценки** $-$ предсказание наиболее вероятной оценки по выбранной дисциплине на основе заранее обученной модели машинного обучения.
- **Прогноз типа сдачи дисциплины** $-$ определение наиболее вероятного типа сдачи дисциплины (первая сдача, пересдача и т.д.) на основе заранее обученной модели машинного обучения.
- **Одиночное прогнозирование** $-$ ввод данных студента через веб-интерфейс.
- **Пакетная обработка** $-$ загрузка CSV-файла с несколькими студентами для массового получения прогнозов.
- **Визуализация результатов** $-$ визуализация прогнозов и статистики в виде графиков.
- **Экспорт результатов** $-$ сохранение предсказаний в формате CSV.

## Используемые технологии

- Язык: Python
- ML: Scikit-learn, LightGBM, CatBoost, Optuna
- Визуализация: Matplotlib
- Веб-интерфейс: Streamlit

## Требования
- Python 3.10 или выше

## Установка

### Клонирование репозитория

```bash
git clone https://github.com/Liliiax/trajectory-project-2026.git
cd trajectory-project-2026
```

### Создание виртуального окружения

#### Linux/macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск приложения

```bash
streamlit run app.py
```
